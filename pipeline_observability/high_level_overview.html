  4. Step 2A: Query with explorations/04_kpi_summary_table.sql
  5. Step 2B: Create missing dlt_event_log_summary table or modify cost queries
  6. Step 2B: Run summaries/05_cost_summary
  7. Step 2B: Query with explorations/05_cost_kpis.sql
  8. Step 2C: Run transformations/raw_columns_metadata.sql in a DLT pipeline
  9. Step 2C: Run summaries/pii_summary.sql in the same DLT pipeline
